# *ACL 論文樣式

此目錄包含最新版本的 *ACL 會議 LaTeX 模板。

## 作者說明

向 *ACL 會議提交的論文必須使用官方的 ACL 樣式模板。

LaTeX 樣式檔案可透過以下方式取得：

- [Overleaf 模板](https://www.overleaf.com/latex/templates/association-for-computational-linguistics-acl-conference/jvxskxpnznfj)
- 在本倉庫中
- 作為 [.zip 檔案](https://github.com/acl-org/acl-style-files/archive/refs/heads/master.zip)

範例請參見 [`acl_latex.tex`](https://github.com/acl-org/acl-style-files/blob/master/acl_latex.tex)。

請遵循 *ACL 會議的一般論文格式指南：

- [論文格式指南](https://acl-org.github.io/ACLPUB/formatting.html)

作者不得修改這些樣式檔案或使用為其他會議設計的模板。

## 出版主席說明

若要為您的會議調整樣式檔案，請 fork 本倉庫並進行必要的更改。最起碼您需要更新會議名稱並重命名檔案。

如果您對模板進行了應傳播到未來會議的改進，請提交 pull request。預先感謝您！

在舊版本的模板中，作者被要求填寫 START 投稿 ID，以便將其蓋印在匿名版本的每一頁頂部。這現在已不再需要，因為現在可以在 START 內自動執行此蓋印。目前，執行此操作的方法是由程序主席寄送電子郵件至 support@softconf.com 並提出請求。

## 修改樣式檔案的說明

- 在 github 中合併 pull request，或 push 到 github
- 從 github git pull 到本地倉庫
- 然後，從您的本地倉庫 git push 到 overleaf 專案
    - Overleaf 專案為 https://www.overleaf.com/project/5f64f1fb97c4c50001b60549
    - Overleaf git url 為 https://git.overleaf.com/5f64f1fb97c4c50001b60549
- 然後，在 overleaf 中點擊 "Submit"（提交），然後選擇 "Submit as Template"（作為模板提交），以請求 overleaf 從 overleaf 專案更新 overleaf 模板
