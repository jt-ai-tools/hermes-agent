# Manim 影片技能 (Manim Video Skill)

使用 [Manim 社群版](https://www.manim.community/) 的數學與技術動畫製作流程。

## 功能介紹

根據文字提示詞創建 3Blue1Brown 風格的動畫影片。代理程式 (agent) 處理完整流程：創意規劃、Python 程式碼生成、渲染、場景拼接以及迭代優化。

## 使用案例

- **概念解釋** — 「解釋類神經網路如何學習」
- **公式推導** — 「動畫展示勾股定理的證明」
- **演算法視覺化** — 「逐步展示快速排序法的運作」
- **數據故事** — 「以動畫展示我們效能指標的前後對比」
- **架構圖** — 「展示我們微服務架構的構建過程」

## 前置需求

Python 3.10+, Manim CE (`pip install manim`), LaTeX, ffmpeg。

```bash
bash skills/creative/manim-video/scripts/setup.sh
```
