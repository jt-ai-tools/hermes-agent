## 1. 視覺主題與氛圍

Framer 的網站是一個具備電影感、工具至上的深色畫布，散發出一種由崇尚工藝的設計師所打造的設計工具的自信。整個體驗沉浸在純黑色中——不是溫暖的炭灰色或舒適的深灰色，而是一個絕對的虛空 (`#000000`)，這使得每個元素、每張截圖、每處排版點綴都像是漂浮在深邃的太空中。這是一個將自家產品 UI 視為英雄藝術的網站，將高保真截圖和互動示範直接嵌入敘事流中。

字體排印是其招牌特色：GT Walsheim 搭配極具侵略性的緊密字距（在 110px 的展示文字上甚至達到 -5.5px），營造出壓縮感、動態感，甚至像蓄勢待發的標題——彷彿承受壓力的文字隨時可能擴張。切換到內文的 Inter 字體非常無縫，並大量使用 OpenType 特性 (`cv01`, `cv05`, `cv09`, `cv11`, `ss03`, `ss07`)，使即便微小的文字也帶有精緻的客製感。Framer Blue (`#0099ff`) 的運用雖節制但果敢——作為連結顏色、邊框裝飾和細微的環狀陰影——在缺乏溫度的黑色背景上劃出一道冷冽、帶電的主線。

整體效果就像是專為網頁設計師打造的夜店：深邃、精準、迷人，且毫不掩飾地以產品為中心。每個區塊的存在都是為了展示工具的能力，網站本身即是概念驗證。

**關鍵特徵：**
- 純黑 (`#000000`) 虛空畫布——絕對的黑暗，不帶暖色調或灰色調
- GT Walsheim 展示字體，具有極致的負字距（110px 時為 -5.5px）
- Framer Blue (`#0099ff`) 作為唯一的強調色——冷冽、帶電、精準
- 藥丸形按鈕（40px–100px 圓角半徑）——互動元素上沒有銳角
- 產品截圖作為英雄藝術——工具本身就是行銷
- 在深色表面上使用 `rgba(255, 255, 255, 0.1)` 的磨砂玻璃按鈕變體
- 在 Inter 字體上廣泛使用 OpenType 特性，以實現精緻的微觀排版

## 2. 色彩配置與角色

### 主要色彩
- **純黑** (`#000000`)：主要背景，定義了 Framer 深色優先識別的虛空畫布
- **純白** (`#ffffff`)：深色表面上的主要文字顏色，強調背景上的按鈕文字
- **Framer Blue** (`#0099ff`)：主要強調色——連結、邊框、環狀陰影、互動高亮

### 次要與強調色
- **柔和銀** (`#a6a6a6`)：次要文字、柔和標籤、深色表面上的淡化描述
- **近乎黑** (`#090909`)：提升的深色表面，用於細微深度分離的陰影環顏色

### 表面與背景
- **虛空黑** (`#000000`)：頁面背景，主要畫布
- **磨砂白** (`rgba(255, 255, 255, 0.1)`)：半透明按鈕背景，深色上的玻璃效果表面
- **細緻白** (`rgba(255, 255, 255, 0.5)`)：略為不透明的磨砂元素，用於懸停 (hover) 狀態

### 中性色與文字
- **純白** (`#ffffff`)：標題文字、高強調性的本文文字
- **柔和銀** (`#a6a6a6`)：本文文字、描述、次要資訊
- **幽靈白** (`rgba(255, 255, 255, 0.6)`)：三級文字，深色表面上的佔位符

### 語意與強調
- **Framer Blue** (`#0099ff`)：連結、互動邊框、焦點環
- **藍光暈** (`rgba(0, 153, 255, 0.15)`)：焦點環陰影，互動元素周圍的細微藍色光圈
- **預設連結藍** (`#0000ee`)：標準瀏覽器連結顏色（在內容區域中節制使用）

### 漸層系統
- 無顯著的漸層使用——Framer 依賴純色平面黑色表面，偶爾使用藍色調的光暈來增加深度
- 產品截圖後方使用極低不透明度的 Framer Blue 營造細微的徑向光暈效果

## 3. 字體排印規則

### 字體系列
- **展示 (Display)**：`GT Walsheim Framer Medium` / `GT Walsheim Medium` — 客製化幾何無襯線體，字重 500。後備字體：`GT Walsheim Framer Medium Placeholder`, 系統無襯線體
- **本文/UI**：`Inter Variable` / `Inter` — 具備豐富 OpenType 特性的可變無襯線體。後備字體：`Inter Placeholder`, `-apple-system`, `system-ui`
- **強調**：`Mona Sans` — GitHub 的開源字體，用於特定元素的超細字重 (100)
- **等寬 (Monospace)**：`Azeret Mono` — 用於程式碼和技術標籤的配套等寬字體
- **圓體 (Rounded)**：`Open Runde` — 用於微標籤的小型圓體配套字體

### 原則
- **壓縮即個性**：GT Walsheim 的極致負字距（110px 時為 -5.5px）是定義性的排版動作——標題感覺蓄勢待發、緊迫且近乎屏息。
- **OpenType 極大主義**：Inter 同時運用了 6 個以上的 OpenType 特性 (`cv01`, `cv05`, `cv09`, `cv11`, `ss03`, `ss07`)，即便在本文大小下也能營造出細微的客製感。
- **展示文字的字重克制**：所有 GT Walsheim 的使用均為字重 500 (Medium)——絕不使用粗體 (Bold) 或常規體 (Regular)。這創造了一種自信但不具侵略性的展示調性。
- **超緊湊行高**：展示文字行高為 0.85，意味著字母在垂直方向上幾乎重疊——這種刻意的密度在遠處閱讀時會更有效果。

## 4. 組件樣式

### 按鈕
- **磨砂藥丸型**：`rgba(255, 255, 255, 0.1)` 背景，黑色文字 (`#000000`)，藥丸形狀（40px 圓角半徑）。這種存在於深色表面上的玻璃效果按鈕——半透明、氛圍感、細緻。
- **純白藥丸型**：`rgb(255, 255, 255)` 背景，黑色文字 (`#000000`)，完整藥丸形狀（100px 圓角半徑），內距 `10px 15px`。主要的行動呼籲 (CTA)——簡潔、在深色背景上形成高對比、不容錯過。
- **幽靈按鈕**：無可見背景，白色文字，僅依靠文字樣式。懸停時顯示細微的磨砂背景。
- **過渡**：基於縮放的動畫（縮放比例為 0.85 的矩陣轉換），用於顯示效果的不透明度過渡。

### 卡片與容器
- **深色表面卡片**：黑色或近乎黑色 (`#090909`) 背景，`rgba(0, 153, 255, 0.15) 0px 0px 0px 1px` 藍色環狀陰影邊框，圓角（10px–15px 半徑）。
- **提升卡片**：多層陰影——`rgba(255, 255, 255, 0.1) 0px 0.5px 0px 0.5px`（細微的頂部高亮）+ `rgba(0, 0, 0, 0.25) 0px 10px 30px`（深邃的氛圍陰影）。
- **產品截圖**：在深色容器中全寬或帶有內距，軟體 UI 預覽使用 8px–12px 圓角。
- **懸停 (Hover)**：Framer Blue 環狀陰影的亮度細微增加，或磨砂表面的亮度轉變。

### 輸入框與表單
- 行銷網站上極少出現表單。
- 輸入欄位遵循深色主題：深色背景、細微邊框、白色文字。
- 焦點狀態：Framer Blue (`#0099ff`) 環狀邊框，`1px solid #0099ff`。
- 佔位符文字使用 `rgba(255, 255, 255, 0.4)`。

### 導覽
- **深色浮動導覽列**：黑色背景帶有磨砂玻璃效果，白色文字連結。
- **導覽連結**：Inter 字體 15px，字重 400，白色文字，懸停時不透明度有細微變化。
- **CTA 按鈕**：藥丸形，白色或磨砂，位於導覽列右端。
- **行動裝置**：縮減為漢堡選單，維持深色主題。
- **黏性行為 (Sticky)**：捲動時導覽列固定在頂部。

### 圖像處理
- **產品截圖作為英雄藝術**：全寬嵌入的 UI 截圖，帶有圓角 (8px–12px)。
- **黑底黑構圖**：截圖放置在黑色背景上，使用細微陰影進行深度分離。
- **16:9 與自定義長寬比**：產品示範填滿其容器。
- **無裝飾性圖像**：所有圖像皆具功能性——展示工具、輸出或工作流程。

### 信任與社交證明
- 客戶標誌和證言在深色表面上使用柔和灰色。
- 極簡裝飾——產品截圖本身即是信任訊號。

## 5. 版面配置原則

### 間距系統
- **基準單位**：8px
- **比例**：1px, 2px, 3px, 4px, 5px, 6px, 8px, 10px, 12px, 15px, 20px, 30px, 35px
- **區塊內距**：巨大的垂直間距（區塊間 80px–120px）。
- **卡片內距**：15px–30px 內部內距。
- **組件間距**：相關元素間 8px–20px。

### 網格與容器
- **最大寬度**：~1200px 容器，居中。
- **欄位模式**：全寬英雄區塊、2 欄特色區塊、單欄產品展示。
- **不對稱版面**：特色區塊通常將文字 (40%) 與截圖 (60%) 配對。

### 留白哲學
- **在黑暗中呼吸**：區塊間慷慨的垂直間距——黑色背景意味著留白呈現為虛空，在內容塊之間創造出戲劇性的停頓。
- **內密外疏**：單個組件構圖緊湊（緊湊行高、壓縮文字），但浮動在慷慨的周圍空間中。
- **產品優先的密度**：截圖區域允許密集且資訊豐富，與稀疏的行銷文字形成對比。

### 圓角比例 (Border Radius Scale)
- **1px**：微型元素，近乎直角的精準邊緣。
- **5px–7px**：小型 UI 元素、圖像縮圖——細微柔化。
- **8px**：標準組件半徑——程式碼塊、按鈕、互動元素。
- **10px–12px**：卡片、產品截圖——舒適的圓角。
- **15px–20px**：大型容器、特色卡片——慷慨的圓角。
- **30px–40px**：導覽藥丸、分頁——顯著的圓角。
- **100px**：完整藥丸形狀——主要 CTA、標籤元素。

## 6. 深度與高度

| 層級 | 處理方式 | 用途 |
|-------|-----------|-----|
| Level 0 (平面) | 無陰影，純黑表面 | 頁面背景、空白區域 |
| Level 1 (環狀) | `rgba(0, 153, 255, 0.15) 0px 0px 0px 1px` | 卡片邊框、互動元素輪廓——Framer Blue 光暈環 |
| Level 2 (內縮) | `rgb(9, 9, 9) 0px 0px 0px 2px` | 用於深色表面細微包圍的近黑環 |
| Level 3 (浮動) | `rgba(255, 255, 255, 0.1) 0px 0.5px 0px 0.5px, rgba(0, 0, 0, 0.25) 0px 10px 30px` | 提升的卡片、浮動元素——細微白色頂邊高亮 + 深邃氛圍陰影 |

### 陰影哲學
Framer 的高度系統與傳統的淺色主題設計相反。Framer 不在淺色背景上使用深色陰影，而是使用：
- **藍色調環狀陰影**，以極低的不透明度 (0.15) 進行包圍——這是一個標誌性動作，為每個帶邊框的元素加上品牌色彩。
- **白色邊緣高亮** (0.5px) 位於提升元素的頂邊——模擬光線照射在頂部表面的效果。
- **深邃氛圍陰影** 用於真正的浮動元素——`rgba(0, 0, 0, 0.25)`，具有較大的擴散範圍 (30px)。

### 裝飾性深度
- **藍光氛圍**：在關鍵互動區域後方使用細微的 Framer Blue (`#0099ff`) 徑向漸層。
- **無背景模糊/毛玻璃效果**：儘管有磨砂按鈕效果，但並沒有大量使用厚重的玻璃模糊——半透明性是透過簡單的 rgba 不透明度實現的。

## 7. 該做與不該做

### 該做 (Do)
- 使用純黑 (`#000000`) 作為主要背景——不是深灰色，也不是炭灰色。
- 在 GT Walsheim 展示文字上應用極致負字距（-3px 到 -5.5px）。
- 保持所有按鈕為藥丸形（40px+ 半徑）——永遠不要使用方形或略帶圓角的按鈕。
- 專門將 Framer Blue (`#0099ff`) 用於互動強調——連結、邊框、焦點狀態。
- 在深色背景上的磨砂玻璃表面使用 `rgba(255, 255, 255, 0.1)`。
- 將 GT Walsheim 維持在字重 500——中等字重即是品牌特色。
- 在 Inter 文字上廣泛使用 OpenType 特性 (cv01, cv05, cv09, cv11, ss03, ss07)。
- 讓產品截圖成為視覺核心——工具本身即是行銷。
- 應用藍色環狀陰影 (`rgba(0, 153, 255, 0.15) 0px 0px 0px 1px`) 用於卡片包圍。

### 不該做 (Don't)
- 使用暖深色背景（不使用 `#1a1a1a`, `#2d2d2d` 或偏褐色的黑色）。
- 在 GT Walsheim 展示文字上應用粗體 (700+) 字重——僅使用 Medium 500。
- 引入除 Framer Blue 以外的其他強調色——這是一個單一強調色的系統。
- 在非互動元素上使用大圓角（卡片使用 10px–15px，只有按鈕使用 40px+）。
- 加入裝飾性圖像、插圖或圖示——產品本身即是插圖。
- 在標題上使用正字距——一切都是壓縮的、負字距。
- 建立沉重的下陰影——深度是透過細微的環和極小的氛圍感來傳達的。
- 在內容區塊後方放置淺色/白色背景——虛空是神聖的。
- 使用襯線體或展示級字重字體——系統僅使用幾何無襯線體。

## 8. 響應式行為

### 斷點 (Breakpoints)
| 名稱 | 寬度 | 關鍵變化 |
|------|-------|-------------|
| 行動裝置 | <809px | 單欄、堆疊的特色區塊、縮減的英雄文字 (62px→40px)、漢堡選單 |
| 平板 | 809px–1199px | 開始出現 2 欄特色區塊、導覽連結部分可見、截圖比例縮小 |
| 桌上型電腦 | >1199px | 完整版面、展開所有連結與 CTA 的導覽、110px 展示英雄區塊、並排特色區塊 |

### 觸控目標
- 藥丸按鈕：最小高度 40px，垂直內距 10px——超過 WCAG 44px 的最小值。
- 導覽連結：15px 文字，具備慷慨的內距以確保觸控易用性。
- 行動裝置 CTA 按鈕：行動裝置上採用全寬藥丸形，便於拇指觸及。

### 折疊策略
- **導覽**：在行動裝置斷點處從完整水平導覽轉換為漢堡選單。
- **英雄文字**：跨斷點從 110px 展示文字縮減至 85px → 62px → ~40px，等比例維持極致負字距。
- **特色區塊**：從並排（文字 + 截圖）在行動裝置上變為垂直堆疊。
- **產品截圖**：在容器內響應式縮放，維持長寬比。
- **區塊間距**：等比例減少——桌面 120px → 行動裝置 60px。

### 圖像行為
- 產品截圖是響應式的，在其容器邊界內縮放。
- 沒有藝術導向 (Art direction) 的變化——不同斷點使用相同的裁切。
- 深色背景確保截圖在任何尺寸下都能保持視覺衝擊力。
- 截圖在使用者捲動進入視野時進行延遲載入 (Lazy-load)。

## 9. 代理提示指南 (Agent Prompt Guide)

### 快速色彩參考
- 主要背景：虛空黑 (`#000000`)
- 主要文字：純白 (`#ffffff`)
- 強調/CTA：Framer Blue (`#0099ff`)
- 次要文字：柔和銀 (`#a6a6a6`)
- 磨砂表面：半透明白 (`rgba(255, 255, 255, 0.1)`)
- 提升環：藍光 (`rgba(0, 153, 255, 0.15)`)

### 組件提示範例
- 「在純黑背景上建立一個英雄區塊，標題使用 110px 白色 GT Walsheim，字距 -5.5px，行高 0.85，以及一個帶有黑色文字的藥丸形白色 CTA 按鈕（100px 半徑）」
- 「在黑色背景上設計一個特色卡片，帶有 1px Framer Blue 環狀陰影邊框 (rgba(0,153,255,0.15))、12px 圓角、白色 Inter 標題 22px 字重 700，以及柔和銀 (a6a6a6) 本文文字」
- 「建立一個黑色背景的導覽列，白色 Inter 文字連結 15px，並以磨砂藥丸按鈕 (rgba(255,255,255,0.1) 背景，40px 半徑) 作為 CTA」
- 「建立一個產品展示區塊，在黑色背景上嵌入全寬截圖，10px 圓角，細微的多層陰影（白色 0.5px 頂部高亮 + rgba(0,0,0,0.25) 30px 氛圍陰影）」
- 「設計一個定價卡片，使用純黑表面、Framer Blue (#0099ff) 強調所選方案邊框、白色文字層級（24px Inter 粗體標題、14px 常規本文），以及純色白色藥丸型 CTA 按鈕」

### 反覆運算指南
在優化使用此設計系統生成的現有畫面時：
1. 一次專注於一個組件——深色畫布讓每個元素都顯得珍貴。
2. 務必驗證 GT Walsheim 標題的字距——極致的負字距是不可妥協的。
3. 檢查 Framer Blue 是否僅出現在互動元素上——絕不作為非連結的裝飾背景或文字顏色。
4. 確保所有按鈕都是藥丸形的——任何直角都會立即破壞 Framer 的美學。
5. 檢查磨砂玻璃表面是否精確為 `rgba(255, 255, 255, 0.1)`——太不透明看起來像錯誤，太透明則會消失。
