# 輸出格式範例

## 章節 (Chapters)

```
00:00 Introduction
02:15 Background and motivation
05:30 Main approach
12:45 Results and evaluation
18:20 Limitations and future work
21:00 Q&A
```

## 摘要 (Summary)

涵蓋影片亮點、主要論點和結論的 5-10 句話概述。以第三人稱、現在式撰寫。

## 章節摘要 (Chapter Summaries)

```
## 00:00 Introduction (2 min)
講者介紹主題 X 並解釋為何它對 Y 很重要。

## 02:15 Background (3 min)
回顧該領域先前的工作，涵蓋方法 A、B 和 C。
```

## 討論串 (Twitter/X Thread)

```
1/ 剛看了一場關於 [主題] 的精彩演講。以下是重點摘要： 🧵

2/ 第一個見解：[重點]。這很重要，因為 [原因]。

3/ 令人驚訝的部分：[意外發現]。大多數人認為 [普遍觀點]，但數據顯示並非如此。

4/ 實用建議：[可執行的建議]。

5/ 完整影片：[URL]
```

## 部落格文章 (Blog Post)

完整文章包含：
- 標題
- 前言段落
- 每個主要主題的 H2 區段
- 關鍵語錄（附帶時間戳記）
- 結論 / 重點摘要

## 語錄 (Quotes)

```
"最重要的不是模型大小，而是數據品質。" — 05:32
"我們發現，規模超過 70B 參數後收益遞減。" — 12:18
```
