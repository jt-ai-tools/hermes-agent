# inference.sh

透過 [inference.sh](https://inference.sh) 平台在雲端執行超過 150 種 AI 應用程式。

**「一把 API 金鑰搞定一切」** — 透過單一帳號存取圖像生成、影片創作、大型語言模型 (LLMs)、搜尋、3D 建模等功能。無需為每個供應商管理獨立的 API 金鑰。

## 可用技能

- **cli**: 透過終端機工具使用 inference.sh CLI (`infsh`)

## 包含內容

- **圖像生成**: FLUX, Reve, Seedream, Grok Imagine, Gemini
- **影片生成**: Veo, Wan, Seedance, OmniHuman, HunyuanVideo
- **大型語言模型 (LLMs)**: Claude, Gemini, Kimi, GLM-4 (透過 OpenRouter)
- **搜尋**: Tavily, Exa
- **3D 建模**: Rodin
- **社群媒體**: Twitter/X 自動化
- **音訊**: 文字轉語音 (TTS)、聲音複製
