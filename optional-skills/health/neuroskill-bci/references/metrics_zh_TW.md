# NeuroSkill 指標定義與解讀指南

> **⚠️ 僅供研究使用：** 所有指標皆為實驗性質，源自消費級硬體 (Muse 2/S)。它們未獲得 FDA/CE 認證，絕不可用於醫療診斷或治療。

---

## 硬體與訊號獲取

NeuroSkill 已針對 **Muse 2** 與 **Muse S** 頭帶進行驗證 (桌面應用程式支援 OpenBCI)，串流頻率為 **256 Hz** (EEG) 與 **64 Hz** (PPG)。

### 電極位置 (國際 10-20 系統)
| 通道 | 電極 | 位置 | 主要訊號 |
|---------|-----------|----------|-----------------|
| CH1 | TP9 | 左耳後 (Mastoid) | 聽覺皮層、語言記憶、咬牙偽影 |
| CH2 | AF7 | 左前額 (Prefrontal) | 執行功能、趨近動機、眨眼 |
| CH3 | AF8 | 右前額 (Prefrontal) | 情緒調節、警覺性、眨眼 |
| CH4 | TP10 | 右耳後 (Mastoid) | 韻律、空間聽覺、非語言認知 |

### 預處理流程
1. **濾波**: 高通 (0.5 Hz)、低通 (50/60 Hz)、陷波濾波 (Notch filter)
2. **頻譜分析**: Hann 視窗化 FFT (512 樣本視窗)、Welch 週期圖
3. **GPU 加速**: 透過 `gpu_fft` 實現約 125 毫秒的延遲

## EEG 頻帶

相對功率值 (所有頻帶總和 ≈ 1.0)：

| 頻帶 | 範圍 (Hz) | 高數值代表 | 低數值代表 |
|------|-----------|------------|-----------|
| **Delta (δ)** | 1–4 | 深層睡眠 (N3)、高振幅偽影 | 清醒、警覺 |
| **Theta (θ)** | 4–8 | 睏倦、REM 開始、創意構思、認知負荷 | 警覺、專注 |
| **Alpha (α)** | 8–13 | 放鬆的清醒狀態、努力時的「Alpha 阻斷」 | 積極思考、焦慮 |
| **Beta (β)** | 13–30 | 積極集中力、解決問題、警覺性 | 放鬆、不專注 |
| **Gamma (γ)** | 30–50 | 高階處理、感知綁定、記憶 | 基準線 |

```json
"bands": {
  "rel_delta": 0.28, "rel_theta": 0.18, "rel_alpha": 0.32,
  "rel_beta": 0.17, "rel_gamma": 0.05
}
```

## 核心綜合評分 (0–1 刻度)

### 專注度 (Focus)
- **公式**: σ(β / (α + θ)) — Beta 波相對慢波的主導地位，經過 Sigmoid 映射
- **> 0.70**: 深度集中、心流狀態、任務沉浸
- **0.40–0.69**: 中度注意力，伴隨些許分心
- **< 0.40**: 分心、疲勞、難以集中

### 放鬆度 (Relaxation)
- **公式**: σ(α / (β + θ)) — Alpha 波的主導地位，經過 Sigmoid 映射
- **> 0.70**: 冷靜、無壓力、副交感神經主導
- **0.40–0.69**: 存在輕微緊張感
- **< 0.30**: 壓力大、焦慮、交感神經主導

### 參與度 (Engagement)
- **0–1 刻度**: 主動的精神投入與動力
- **> 0.70**: 精神投入、有動力、積極處理中
- **0.40–0.69**: 被動參與
- **< 0.30**: 感到無聊、脫離狀態、自動駕駛模式

### 冥想 (Meditation)
- **綜合指標**: 結合 Alpha 波提升、身體靜止度 (IMU) 以及 HRV 一致性
- **> 0.70**: 深度冥想狀態
- **< 0.30**: 活躍、非冥想狀態

### 情緒 (Mood)
- **綜合指標**: 源自 FAA、TAR 與 BAR
- **> 0.60**: 正向情感、趨近動機
- **< 0.40**: 情緒低落、退縮傾向

### 認知負荷 (Cognitive Load)
- **公式**: (P_θ_frontal / P_α_temporal) · f(FAA, TBR) — 工作記憶使用情況
- **> 0.70**: 工作記憶接近極限、複雜處理中
- **0.40–0.69**: 中度精神負擔
- **< 0.40**: 任務簡單或自動化
- **解讀**: 高負荷 + 高專注 = 有效的挑戰。高負荷 + 低專注 = 感到不知所措。

### 睏倦度 (Drowsiness)
- **綜合指標**: 加權後的 TAR + TBR + 下降的頻譜重心 (Spectral Centroid)
- **> 0.60**: 睡眠壓力累積，存在微睡眠風險
- **0.30–0.59**: 輕微疲勞
- **< 0.30**: 警覺

## EEG 比率與頻譜指標

| 指標 | 公式 | 解讀 |
|--------|---------|----------------|
| **FAA** | ln(P_α_AF8) − ln(P_α_AF7) | 前額 Alpha 不對稱性。正值 = 趨近/正向情感。負值 = 退縮/抑鬱。 |
| **TAR** | P_θ / P_α | Theta/Alpha 比率。 > 1.5 = 睏倦或分心。 |
| **BAR** | P_β / P_α | Beta/Alpha 比率。 > 1.5 = 警覺、投入的認知。也可能代表焦慮。 |
| **TBR** | P_θ / P_β | Theta/Beta 比率。ADHD 生物標記。健康 ≈ 1.0, 提升 > 1.5, 臨床 > 3.0。 |
| **APF** | argmax_f PSD(f) 範圍 [7.5, 12.5] Hz | Alpha 峰值頻率。典型為 8–12 Hz。越高 = 認知處理越快。隨年齡/疲勞減慢。 |
| **SNR** | 10 · log₁₀(P_signal / P_noise) | 訊噪比。 > 10 dB = 清晰, 3–10 dB = 可用, < 3 dB = 不可靠。 |
| **Coherence** | 半球間相干性 (0–1) | 大腦半球間的皮層連接性。 |
| **Mu Suppression** | 感覺運動皮層抑制指數 | 運動或運動想像時數值較低。 |

## 複雜度與非線性指標

| 指標 | 說明 | 健康範圍 |
|--------|-------------|---------------|
| **排列熵 (PE)** | 時間複雜度。接近 1 = 極度不規則。 | 意識指標 |
| **Higuchi 分形維度 (HFD)** | 波形自相似性。 | 清醒: 1.3–1.8; 越高 = 越複雜 |
| **DFA 指數** | 長程相關性。 | 健康: 0.6–0.9 |
| **PSE** | 功率譜熵。接近 1.0 = 白噪音。 | 越低 = 大腦狀態越有條理 |
| **PAC θ-γ** | 相位-振幅耦合 (Theta-Gamma)。 | 工作記憶機制 |
| **BPS** | 頻帶功率斜率 (1/f 頻譜指數)。 | 越陡 = 抑制佔主導 |

## 意識指標

衍生自上述非線性指標：

| 指標 | 刻度 | 解讀 |
|--------|-------|----------------|
| **LZC** | 0–100 | Lempel-Ziv 複雜度代理指標 (PE + HFD)。 > 60 = 清醒。 |
| **清醒度 (Wakefulness)** | 0–100 | 睏倦度綜合指標的反函數。 |
| **整合度 (Integration)** | 0–100 | 皮層整合 (相干性 × PAC × 頻譜熵)。 |

狀態閾值：≥ 50 綠色, 25–50 黃色, < 25 紅色。

| 指標 | 說明 | 正常 / 綠色範圍 |
|--------|-------------|---------------------|
| **HR** | 心率 (bpm) | 55–90 (綠), 45–110 (黃), 否則為紅 |
| **RMSSD** | 主要迷走神經張力指標 (ms) | > 50 ms 健康, < 20 ms 壓力大 |
| **SDNN** | HRV 時域變異性 (ms) | 越高越好 |
| **pNN50** | 副交感神經指標 (%) | 越高代表副交感神經活動越強 |
| **LF/HF 比率** | 交感-迷走神經平衡 | > 2.0 = 壓力大, < 0.5 = 放鬆 |
| **壓力指數 (Stress Index)** | Baevsky SI 公式 | 0–100 綜合指標。原始值 > 200 = 強烈壓力 |
| **SpO₂ 估計值** | 血氧飽和度 (未校準) | 95–100% 正常 (僅供研究) |
| **呼吸率** | 每分鐘呼吸次數 | 12–20 正常 |

## 動作與偽影檢測

| 指標 | 說明 |
|--------|-------------|
| **靜止度 (Stillness)** | 0–1 (1 = 完全靜止)。源自 IMU 加速度計/陀螺儀。 |
| **眨眼計數** | 檢測到的眨眼次數 (AF7/AF8 的大尖峰)。正常：15–20 次/分。 |
| **咬牙計數** | TP9/TP10 檢測到的高頻 EMG 爆發 (> 30 Hz)。 |
| **點頭計數** | 透過 IMU 檢測到的點頭動作。 |
| **搖頭計數** | 透過 IMU 檢測到的搖頭動作。 |
| **頭部俯仰/翻滾 (Pitch/Roll)** | 源自 IMU 的頭部方向。 |

## 睡眠分期

基於 5 秒一期，使用相對頻帶功率比率與 AASM 啟發式演算法：

| 階段 | 代碼 | EEG 特徵 | 功能 |
|-------|------|---------------|----------|
| 清醒 (Wake) | 0 | Alpha 波佔主導, BAR > 0.8 | 意識覺知 |
| N1 | 1 | Alpha → Theta 過渡 | 進入淺層睡眠 |
| N2 | 2 | 睡眠梭狀波 (Spindles)、K-複合波 | 記憶鞏固 |
| N3 (深層) | 3 | Delta > 一期的 20%, DTR > 2 | 深層修復性睡眠 |
| REM | 4 | 活躍的 EEG, 高 Theta, 低 Delta | 情緒處理、作夢 |

### 健康成年人目標 (約 8 小時睡眠)
- **N3 (深層)**: 總睡眠的 15–25%
- **REM**: 20–25%
- **睡眠效率**: > 85%
- **入睡潛伏期 (Sleep Onset Latency)**: < 20 分鐘

## 綜合狀態模式

| 模式 | 關鍵指標 | 解讀 |
|---------|-------------|----------------|
| **心流狀態 (Flow)** | 專注度 > 0.75, 參與度 > 0.70, 認知負荷 0.50–0.70, 心率穩定 | 最佳表現區 — 應予以保護 |
| **精神疲勞** | 專注度 < 0.40, 睏倦度 > 0.60, TBR > 1.5, Theta 波提升 | 需要休息或中斷 |
| **焦慮** | 放鬆度 < 0.30, 心率提升, 高 Beta, 高 BAR, 壓力指數高 | 適合進行放鬆干預 |
| **巔峰警覺** | 專注度 > 0.80, 參與度 > 0.70, 睏倦度 < 0.20 | 處理困難任務的最佳時機 |
| **恢復** | 放鬆度 > 0.70, HRV (RMSSD) 上升, Alpha 波佔主導 | 整合階段，僅適合輕度任務 |
| **創意模式** | 高 Theta, 高 Alpha, 低 Beta, 中度專注 | 構思階段 — 不要強求結構化 |
| **退縮** | FAA < 0, 情緒低落, 低參與度 | 需要趨近動機 (Approach motivation) |

## ZUNA 嵌入 (Embeddings)

NeuroSkill 使用 **ZUNA 神經編碼器** 將 5 秒的 EEG 一期轉換為 **128 維向量**，並存儲於 HNSW 索引中：
- **搜尋**: 亞毫秒級的近似最近鄰查詢 (ANN)
- **UMAP**: GPU 加速的 3D 投影，用於視覺化比較
- **存儲**: 本地 SQLite + HNSW 索引，位於 `~/.skill/YYYYMMDD/`
