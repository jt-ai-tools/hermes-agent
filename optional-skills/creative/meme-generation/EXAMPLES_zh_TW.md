# 迷因生成範例

## 範例 1：凌晨 2 點的除錯

**主題：** 凌晨 2 點在正式環境除錯
**模板：** this-is-fine（這一切都很好）

```bash
python generate_meme.py this-is-fine /tmp/meme.png "正式環境掛掉了" "這一切都很好"
```

## 範例 2：開發者的優先順序

**主題：** 在撰寫測試與發布功能之間做選擇
**模板：** drake（德瑞克）

```bash
python generate_meme.py drake /tmp/meme.png "撰寫單元測試" "直接部署到正式環境"
```

## 範例 3：考試壓力

**主題：** 期末考準備
**模板：** two-buttons（兩個按鈕）

```bash
python generate_meme.py two-buttons /tmp/meme.png "什麼都讀" "睡覺" "午夜時分的我"
```

## 範例 4：逐步升級的解決方案

**主題：** 修復 CSS 錯誤
**模板：** expanding-brain（擴張的大腦）

```bash
python generate_meme.py expanding-brain /tmp/meme.png "閱讀文件" "Stack Overflow" "所有東西都加 !important" "刪除樣式表"
```

## 範例 5：辛辣觀點

**主題：** Tabs（縮排）對決 Spaces（空格）
**模板：** change-my-mind（改變我的想法）

```bash
python generate_meme.py change-my-mind /tmp/meme.png "Tabs 只是比較肥的空格"
```
